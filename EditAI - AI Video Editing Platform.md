# EditAI - AI Video Editing Platform

AI-powered video editing platform that generates structured edit plans from natural language commands.

## Project Structure

```
editai/
├── frontend/          # React + TypeScript
│   ├── src/
│   ├── index.html
│   ├── package.json
│   ├── vite.config.ts
│   └── tsconfig.json
├── backend/           # Node.js + Express
│   ├── src/
│   │   └── server.ts
│   ├── package.json
│   ├── tsconfig.json
│   └── dist/
├── .gitignore
└── README.md
```

## Quick Start

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm run dev
```

## Deployment

- **Frontend**: Vercel
- **Backend**: Railway
